🚀 Quick Setup Guide

  1. 1
    Create Firebase Project
    Go to Firebase Console and create a new project or select existing one.
  2. 2
    Add Web App
    In Project Settings > General, scroll to "Your apps" section and click "Add app" > Web (< />). Give it any name like "Vocab SRS Extension".
  3. 3
    Copy Configuration
    Firebase will show you a config object. Copy the values to the form below.
  4. 4
    Enable Firestore
    Go to Firestore Database and click "Create database". Choose "Start in test mode" for easy setup.
  5. 5
    Test Connection
    Fill the form below and click "Test Connection" to verify everything works!
