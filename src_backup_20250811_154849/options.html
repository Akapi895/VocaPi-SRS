Vocab SRS - Spaced Repetition Vocabulary Learning

Version: 1.0.0

Algorithm: SM-2 (SuperMemo 2)

Dictionary API: dictionaryapi.dev (Free)

How it works:

  • Highlight any English word on any webpage
  • Click "Add to Dictionary" to save it with meaning
  • Use the extension popup to review due words
  • Rate your knowledge (0-5) to optimize review intervals

Privacy:

  • No user data is sent to external servers (except dictionary lookups)
  • Only the selected word is sent to dictionary API
  • All vocabulary data is stored locally in your browser
  • No tracking, analytics, or user identification
